## 📁 Folder Structure

```
portfolio/
├── index.html              ← Main website (all code is here — single file)
├── Oindree_Bal_CV.pdf      ← YOUR CV — add this file here
├── photo.jpg               ← YOUR profile photo — add this file here (optional)
```

Everything is in a single `index.html` file for simplicity. All CSS and JavaScript are embedded, so there are no build steps, no dependencies, and no Node.js required.
